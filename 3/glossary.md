# Глосарій термінів (Ubiquitous Language)

Цей документ містить офіційні визначення ключових термінів проєкту **Clothify Store**.

| Термін                   | Визначення | Приклад у системі |
|:-------------------------| :--- | :--- |
| **Товар (Item)**         | Одиниця одягу з характеристиками (тканина, країна) та ціною. | `Світшот 'Retro'` |
| **Категорія (Category)** | Логічне групування товарів для фільтрації. | `Верх`, `Низ` |
| **Кошик (Cart)**         | Тимчасове сховище обраних товарів користувача. | Об'єкт зі списком `CartItem` |
| **Кількість (Quantity)** | Показник одиниць одного товару в кошику (x2/x3). | `qty: 3` |
| **Аватар (Avatar)**      | Зображення профілю користувача. | `1_photo_2026.jpg` |
| **Користувач (User)**    | Зареєстрована особа з доступом до кабінету та кошика. | `kisildarina@gmail.com` |
| **Відгук (Review)**      | Публічне повідомлення авторизованого користувача про товар. | "Дуже м'яка тканина!" |

---

## Особливості використання термінів у контекстах

1. **Відгук (Review):** Це точка перетину контекстів. **User Context** надає ім'я та фото автора, а **Catalog Context** прив'язує текст до конкретного товару.
2. **Товар (Item):** У контексті відгуків виступає об'єктом оцінювання.
